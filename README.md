# Login & Sign Up Page

A **responsive** and **modern** Login & Sign Up page built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates a smooth toggle between Login and Sign Up forms with stylish social icons and animations.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome for icons

---

## Features

- Fully responsive design for desktop and mobile
- Smooth toggle animation between Login and Sign Up forms
- Modern and clean UI
- Social media login icons (Google, Facebook, GitHub, LinkedIn)
- Easy to customize colors, fonts, and layout
- Simple JavaScript functionality for switching forms
